# Code of Conduct

We are committed to providing a welcoming, respectful community for everyone
participating in MergeRisk. Be constructive, assume good intent, and do not
harass, discriminate against, or disclose private information about others.

Report unacceptable behavior privately through the repository's security
contact path. Maintainers may remove, edit, or reject contributions or comments
that violate this Code of Conduct.

This policy is informed by the
[Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
