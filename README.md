# MergeRisk Action Documentation Bundle

This bundle describes the first build of MergeRisk: a GitHub Action that posts a single pull request risk report for teams using AI-assisted development.

Read in this order:

1. `01-product-spec.md` - product scope, positioning, MVP behavior, and non-goals.
2. `docs/superpowers/plans/2026-06-29-mergerisk-action.md` - implementation plan with task-by-task build steps.
3. `02-agent-task-backlog.md` - issue-sized tasks suitable for small-model delegation.

The MVP is intentionally not a SaaS. It is a distributable GitHub Action with bring-your-own-model-key support.
