# Support

For bugs and feature requests, open a GitHub issue and include:

- The MergeRisk version or commit SHA in use.
- The workflow event and a minimal, redacted workflow excerpt.
- The relevant Action log output, with tokens, API keys, and patch content
  removed.

MergeRisk is an advisory pull-request risk tool. For a security vulnerability,
do not open a public issue; use the private reporting process in
[SECURITY.md](SECURITY.md).
